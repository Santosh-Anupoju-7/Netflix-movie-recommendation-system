# 🎬 Netflix Movie Recommendation System

A Machine Learning project that recommends movies to users based on their previous ratings using the **SVD (Singular Value Decomposition)** algorithm. The project is implemented in **Python** using **Google Colab**.

---

## 📌 Project Overview

The Netflix Movie Recommendation System uses collaborative filtering to predict user preferences and recommend movies they are likely to enjoy. It analyzes historical movie ratings and generates personalized recommendations.

---

## 🚀 Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Collaborative Filtering
- SVD Recommendation Model
- Model evaluation using RMSE
- Personalized movie recommendations

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Surprise

---

## 📂 Dataset

The project uses movie and user rating datasets to build the recommendation system.

---

## 📊 Workflow

1. Import required libraries
2. Load the dataset
3. Clean and preprocess the data
4. Perform exploratory data analysis
5. Build the SVD recommendation model
6. Train and evaluate the model
7. Generate personalized movie recommendations

---

## 📈 Model Evaluation

The recommendation model is evaluated using:

- RMSE (Root Mean Square Error)
- Cross Validation

---

## 📷 Sample Output

The notebook generates personalized movie recommendations based on user ratings.
